경험

현재 멋쟁이사자처럼 프론트엔드 스쿨 교육에 참여하고 있습니다. 이전 대학수업을 들으면서는 얇고 폭넓은 기술과 CS기반 지식을 갈고 닦았다면, 이번 교육을 통해 프론트엔드 개발자로서 빈틈을 채워나가는 경험을 하고 있습니다. HTML/CSS와 JavaScript뿐 아니라, React 라이브러리, Firebase 등의 교육을 통해 프론트엔드 기술을 탄탄하게 쌓고 있습니다.

또한 대학생활간 수강한 컴퓨터네트워크, 컴퓨터구조, 운영체제, 알고리즘 등의 과목을 통해 CS기반지식을 터득하여왔고, 수 차례의 프로젝트를 통해서 팀과 협업하는 경험을 쌓아왔습니다.

기술스택

  • HTML/CSS : 웹 애플리케이션을 구축하는데에 불편함 없이 사용가능한 정도입니다.
  • JavaScript : 생각하는대로 코드를 작성할 수 있으나, 효율성을 고려하려면 시간이 좀 더 걸리는 수준입니다.
  • React, Bootstrap : 과거 프로젝트 경험시 일부 적용한 경험이 있습니다.
  • Tailwind : 익숙한 클래스들은 편하게 사용가능하며, 그렇지 않은경우는 cheat sheet을 참고하면서 사용가능합니다.
  • Git : 기본적인 clone, pull, add, commit, push 가 익숙하며, merge와 fetch의 차이점을 숙지하고 있습니다. 과거 프로젝트 경험으로 merge, pull request 등 무리없이 가능합니다.
